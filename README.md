# SuperCube

A hypercasual 3D action runner built in Unity.

## Gameplay

The player stands on one of three platforms (left, middle, right) while enemies charge in from portals above. Swipe or use arrow keys to change lanes, jump, slam, and shoot your way through endless levels.

- **Swipe left / right** — change lane
- **Swipe up** — jump
- **Swipe down** — slam (destroys nearby enemies)
- **Tap** — shoot

Destroy all 3 portals or kill 10 enemies to drop to the next level. All 3 platforms destroyed = game over.
